## Notes Processing Preferences

### Language and Style
- **Только русский язык** — всегда
- Прямо, конкретно, без воды и академизма
- Простые слова для сложных вещей — как объясняет сам Евгений

### Structure
Preferred format:
```
Context → Insight → Implication → Next Action
```

### Prioritization
- **Срочное**: клиентский дедлайн, деньги прямо сейчас → p1-p2
- **Важное**: личный бренд, контент, обучение → p2-p3
- **Стратегическое**: инвестиции, долгосрочные проекты → p3-p4

### СДВГ-контекст
Всегда разбивать большие задачи на конкретные мелкие шаги.
Не "разобраться с ремонтом" → а "позвонить 3 подрядчикам и получить сметы".

## Anti-Patterns (AVOID)

When processing entries DO NOT:
- Создавать абстрактные заметки без Next Action
- Писать задачи типа "подумать о..." — конкретизировать\!
- Дублировать контент без синтеза
- Создавать списки без приоритетов
- Игнорировать СДВГ-контекст — большие задачи всегда дробить

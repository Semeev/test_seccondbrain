# Монетизация бота @Ersaly_bot

> Идея зафиксирована 24.02.2026. Статус: концепция, не реализована.

## Суть

Бот для отслеживания расходов голосом с мультивалютой (KZT/UZS/USD) — потенциально полезен широкой аудитории СНГ. Но сам по себе не уникален. Главная ценность — **лид-магнит для личного бренда и консалтинга**, а не самостоятельный бизнес.

---

## Целевая аудитория
- Женщины/семьи с расходами в нескольких валютах (СНГ)
- Мигранты и экспаты (Казахстан, Узбекистан)
- Малый бизнес — мелкие расходы голосом без Excel

## Варианты монетизации

### 1. Подписка
- $2–5/мес (990–2500 тг) за личный трекер
- Пробный период 7 дней бесплатно
- При 100 юзерах → $200/мес, прибыль ~$190/мес

### 2. Белый лейбл для бизнеса
- Ресторан/клиника берёт бот под свой бренд
- Сотрудники пишут расходы → владелец видит отчёт
- $50–200/мес за настройку и поддержку

### 3. Семейный тариф
- Муж + жена в одном боте, общий баланс
- Технически уже поддерживается (несколько user_id)

### 4. Horeca-инструмент
- Продавать заведениям как учёт мелкой кассы
- "Бариста говорит боту → владелец видит расходы за день"

## Система подписки (как работает)

```
Юзер платит → бот записывает paid_until →
каждый день проверяет → просрочил → доступ закрыт
```

Таблица в БД:
```
users: telegram_id | paid_until | status (active/expired/trial)
```

### Приём оплаты
| Способ | Когда использовать |
|---|---|
| Вручную (карта) | до 50 юзеров — проще всего |
| Telegram Stars | легко подключить, без банков |
| Kaspi Pay | Казахстан, нужно ИП |
| Click/Payme | Узбекистан, нужно юрлицо |
